# GDF_AI_LabGame
Unity simulations illustrating different approaches to using AI in video games. It includes:

1) Patrolling AI
2) Distance Check AI
3) Simple NavMesh AI with Labyrinth
4) Navmesh Step Height Simulation
5) Navmesh AI that follows the player
6) Conditional (State - based) Navmesh AI
7) Simple Finite State Machine (As a concept)
